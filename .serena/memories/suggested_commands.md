# 常用开发命令

- `npm run start`：启动 Expo 开发服务器（支持 iOS/Android/Web）
- `npm run android`：启动并运行 Android 端
- `npm run ios`：启动并运行 iOS 端
- `npm run web`：启动 Web 端
- `npm run lint`：运行 ESLint 进行代码检查
- `npm run reset-project`：清理/重置本地项目状态（见 scripts/reset-project.js）

# 运行前置
- Node.js + npm
- Expo CLI
- Android/iOS 模拟器或真实设备（可选）
